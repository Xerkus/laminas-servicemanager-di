laminas-servicemanager-di

laminas-di integration with laminas-servicemanager

$ composer require laminas/laminas-servicemanager-di

Installation

Install via composer: 


$ composer require laminas/laminas-servicemanager-di

If you are using the laminas-component-installer, you're done!

If not, you will need to add the component as a module to your application. Add the entry 'Laminas\ServiceManager\Di' to your list of modules in your application configuration (typically one of config/application.config.php or config/modules.config.php).
